## Note Numbers
This website helps you memorize the numbers associatied to the notes in a scale.

### How to use it
1.- Select a key that you want to memorize
2.- Once you click Start, you're given 1 minute to memorize the notes and their numbers
3.- Once you start, a card with a random note or a random number is shown. After 10 seconds it reveals the answer
4.- Another card is then shown and it loops endlessly until you select another key or cancel
